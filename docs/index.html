What & Why

Coffee is a major driver of tropical deforestation, but quantifying how much forest loss is coffee-driven requires combining multiple satellite datasets. This system intersects Hansen forest loss records (2001–2024) with FDP coffee probability maps to identify hotspots, then attributes all deforestation by replacement land cover — not just coffee.

How It Works

  1. Sentinel-1/2 composites at 10m from GEE (annual dry-season medians, 2019–2024)
  2. 19-feature stack — spectral indices, SAR backscatter, elevation, temporal change
  3. Hotspot detection — Hansen loss ∩ FDP coffee > 50% → contiguous polygons
  4. Attribution — classify ALL loss by current land cover (coffee / crops / built / regrowth)
  5. ML validation — RF/XGBoost with cross-AOI holdout and S1/S2 ablation
  6. Reports — interactive maps with year slider, attribution charts, LLM narrative agents
